
  # Job Creation Flow

  This is a code bundle for Job Creation Flow. The original project is available at https://www.figma.com/design/e3wFQsPGHfvHpuohRuekr1/Job-Creation-Flow.

  ## Running the code

  From this folder (`Job Creation Flow`):

  ```bash
  npm install
  npm run dev
  ```

  Or from the parent folder (`New job creation flow`):

  ```bash
  npm install
  npm run dev
  ```

  Then open **http://localhost:5173** in your browser.
